Ciência de dados em R

Capítulo 7 Arquivos de outros softwares

Para ler arquivos gerados por outros softwares, como SPSS, SAS e STATA, você pode usar as funções do pacote haven. Este pacote faz parte do tidyverse e é um wrapper da biblioteca ReadStat, escrita em C.

library(haven)

imdb_sas <- read_sas("data/imdb.sas7bdat")
imdb_spss <- read_spss("data/imdb.sav")
imdb_dta <- read_dta("data/imdb.dta")

É possível salvar ou escrever bases em SAS e STATA com suas respectivas funções write_.

write_dta(mtcars,'data/matcars.dta')

Quando importamos arquivos gerados pelo SAS SPSS ou STATA para o R, os rótulos de uma variável podem não ser importados de forma fiel. O pacote haven tem uma soluçao para este problema.

x <- labelled(imdb_dta$cor,c(C = 'Color',B = 'Black and White'))

labelled() adiciona rótulos à valores de uma variável. Para verificar quais são estes rótulos, podemos usar a função print_labels().

print_labels(x)
## 
## Labels:
##            value label
##            Color     C
##  Black and White     B

Agora que já definimos os missings “especiais”, podemos transformá-los no missing padrão do R, representado pelo símbolo NA.

x1 <- zap_missing(x1)

Existem outras funções zap_ interessantes no pacote. Após rotular os valores do vetor, podemos convertê-los, por exemplo, em fator. Para isso, usamos uma função do pacote haven. A função base as.factor() também pode ser usada, mas observe que os rótulos não serão considerados.

x_base <- base::as.factor(x)
levels(x_base)
## [1] ""                "Black and White" "Color"
x_factor <- haven::as_factor(x)
levels(x_factor)
## [1] ""  "B" "C"
